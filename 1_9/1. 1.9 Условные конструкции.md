# Условная конструкция if

Конструкция `if` принимает условие - выражение, которое возвращает значение типа bool. Если это условие истинно, то
выполняется последующий блок инструкций:

```go
package main

import "fmt"

func main() {
	a := 6
	b := 7
	if a < b {
		fmt.Println("a меньше, чем b")
		 
	}
}
```
