# Числа с плавающей точкой

Числа с плавающей точкой — это числа, которые включают дробную часть, например: 1.234, 123.4, 0.00001234. Их внутреннее
представление в компьютере довольно сложное, но для повседневного использования это не столь важно. Достаточно помнить
несколько ключевых моментов:

1. `Числа с плавающей точкой не обладают абсолютной точностью`.
   Некоторые значения невозможно представить точно. Например, результат вычисления 1.01 - 0.99 будет равен
   0.020000000000000018 — это число очень близко к ожидаемому, но всё же не идентично.

2. `Размер числа влияет на его точность`.
   Как и у целых чисел, числа с плавающей точкой имеют фиксированный размер: 32 бита или 64 бита. Чем больше размер, тем
   больше цифр можно использовать для вычислений, что повышает точность.

Кроме чисел, существуют и специальные значения, которые могут встречаться при работе с числами с плавающей точкой:

- `NaN` (Not a Number) — например, результат операции вроде `0/0`.
- Положительная и отрицательная бесконечность `(+∞ и −∞)`.
  В Go определены два основных типа для работы с вещественными числами:

- `float32` — числа с одинарной точностью.
- `float64` — числа с двойной точностью.
  Также в Go есть два типа для работы с комплексными числами, которые имеют мнимую часть:

- complex64
- complex128

Для большинства задач с вещественными числами достаточно использовать `float32`. Однако если вам нужна более высокая
  точность, следует выбрать `float64`.
